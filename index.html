Use monitorEvents() to log events being fired
Click me
monitorEvents( document.body, 'click' )
Ctrl+click a CSS rule or property to go to it in the source. Use {} to expand.
Note: it will only display the context in which the rule is generated, therefore if it is built with a mixin, it will goto the mixin rather than where the mixin was included.
Use copy() to set your clipboard value. Can be used with objects.
In the Sources tab, you can drag and drop a file directly into your editor.
In the Elements > Styles pane, click a color value preview box to open a colorpicker. While the color picker is opened, an eyedropper tool is available while hovering over any contents in the window.
Use console.profile() and console.profileEnd() to dynamically call the devtools profiler.

console.profile('test');

setTimeout(function(){
	console.profileEnd('test');
}, 3000);
